# 自然语言示例

## 中文说明

### 通用检查

- `检查这个 USD`
- `帮我看这个资产有没有明显问题`

### 引用与贴图

- `检查引用和贴图`
- `帮我确认依赖是否完整`

### 结构与 Isaac Sim

- `检查这个资产的 Isaac Sim 结构`
- `帮我看层级和组件语义是否合理`

### 业务口吻

- `按家具和摆件 Stage 1 检查`
- `这个资产能不能进入静态家具 SimReady 第一阶段`
- `检查这个装饰道具是否适合 Stage 1`
- `这个资产能不能先当静态资产交付`
- `这个资产适不适合做碰撞`
- `这个机器人资产适不适合抓取和移动`

## English

### General Validation

- `check this USD asset`
- `validate this asset`

### References and Textures

- `check references and textures`
- `verify external dependencies`

### Structure and Isaac Sim

- `check Isaac Sim structure`
- `validate hierarchy and component semantics`

### Business-Language Prompts

- `validate this as static furniture and decor props`
- `run the Stage 1 furniture check`
- `can this furnishing enter the Stage 1 SimReady flow`
- `can this be used as a static asset`
- `is this asset good for collision`
- `is this robot asset suitable for grasping and moving`
